# NIAKO — Site vitrine

Site web de NIAKO, entreprise de logiciels sur mesure basée à Abidjan, Côte d'Ivoire.

## Stack

- React 19 + Vite 8
- Tailwind CSS v3
- Framer Motion
- @lottiefiles/dotlottie-react

## Développement local

```bash
cd niako-site
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Déploiement sur Vercel

1. Importer le repo `NabOuat/niako` sur [vercel.com](https://vercel.com)
2. **Root Directory** → `niako-site`
3. Framework preset → **Vite** (auto-détecté)
4. Build command → `vite build`
5. Output directory → `dist`

Aucune variable d'environnement requise.
